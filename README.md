# 🏃 Marathon Skills — Web Version

Веб-версия приложения Marathon Skills, точно воспроизводящая C# WPF-приложение.

## Функционал

- **Главная страница** — статистика участников, обратный отсчёт до марафона (15 июня), значок темпа
- **Регистрация** — валидация полей, форматирование телефона, индикатор email, шкала надёжности пароля, загрузка фото
- **Калькулятор BMI** — анимированная дуга, переключение пол м/ж, анимированная шкала, рекомендации
- **Участники** — таблица с поиском, фильтрами по роли/BMI, сортировкой по 9 критериям
- **4 темы** — Ocean Blue, Midnight Dark, Forest Green, Sunset Run

## Деплой на Vercel

1. Загрузите репозиторий на GitHub (public)
2. Зайдите на [vercel.com](https://vercel.com) → New Project
3. Импортируйте репозиторий → Deploy
4. Получите ссылку вида `yourproject.vercel.app`

## Технологии

- Чистый HTML + CSS + JavaScript (без фреймворков)
- Google Fonts: Rajdhani + Inter
- Один файл `index.html` — работает статически
